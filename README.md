# WOW CART – Online Food Ordering System

## 📌 Project Overview
WOW CART is a Java-based console application that allows users to browse food items, place orders, calculate the total bill, generate a unique Order ID, and print a detailed bill.

## ✨ Features
- View Food Menu
- Place Food Orders
- Calculate Total Bill
- Generate Unique Order ID
- Display Date & Time
- Menu-Driven Console Interface

## 🛠 Technologies Used
- Java
- Object-Oriented Programming (OOP)
- ArrayList
- Scanner
- LocalDateTime

## ▶️ How to Run
1. Clone the repository.
2. Open the project in any Java IDE (IntelliJ IDEA, Eclipse, or VS Code).
3. Run `Main.java`.
4. Follow the on-screen menu.
